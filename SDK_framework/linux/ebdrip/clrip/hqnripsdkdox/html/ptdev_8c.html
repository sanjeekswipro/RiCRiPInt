static int32 getNextPageNumber (  )  [static]

Calculate the next page number to print, taking into account the PageRange device parameter (if set).

If no PageRange is set then all pages will be printed in order. PageRange can be set according to the XPS PrintSchema. Examples could be "1-5", "1-2,4-5", "1,2,5", "5,3,1", "5-1" (in this latter case pages 5, 4, 3, 2, and 1 are printed in that order).

The PageRangeGroupIndex and PageRangeGroupOffset device parameters are automatically updated to keep track of state. PageRangeGroupIndex is a zero-based index into a range 'group'. In the case of "1,4-5" the first group is "1" and the second is "4-5". PageRangeGroupOffset is a zero-based page offset into the group currently being printed.

It is possible to obtain the full sequence of pages specified in PageRange without actually having do any RIPping. First set the PageRange (which will automatically reset PageRangeGroupIndex and PageRangeGroupOffset), and then examine the NextPage parameter. Then increment PageRangeGroupOffset and examine NextPage, repeating until NextPage returns XPSPT_PAGES_NOMORE (-2). Note that PageRangeGroupOffset could be reset by quizzing NextPage, so increment whatever value it is currently set as.
