SWMSG_RR_ELEMENT_UPDATE_HITS Struct Reference

#include <swevents.h>

Data Fields

Detailed Description

Increment or decrement the hit count for the element identified by id by the amount given in hits_delta - whether to add or subtract to the hits total is determined by the raise flag.

Field Documentation

void* connection

Opaque handle to the connection to which this event is relevant. All handlers whose connection handle doesn't match should immediately return SW_EVENT_CONTINUE.

unsigned int hits_delta

Number of hits to add to or subtract from the overall hits remaining count.

uint8* id

Unique ID, length RR_ELEMENT_ID_LENGTH.

HqBool raise

Whether to raise or lower the hit count.

sw_tl_ref timeline

Identifies the job containing the pages.
