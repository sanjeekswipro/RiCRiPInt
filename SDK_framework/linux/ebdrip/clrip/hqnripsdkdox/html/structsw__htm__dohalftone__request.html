Detailed Description

Structure describing what the RIP wants sw_htm_api::DoHalftone() to do.

Field Documentation

unsigned int band_num

The ordinal number of the band being processed, within the page.

int channel_id

The index within the separation of the first or only channel. This corresponds to the channel field of sw_htm_colorant_info. This is principally to help screening modules differentiate one channel from another, e.g., when the same halftone instance is being used to screen several channels independently of each other, such as will often be the case when screening spot colors using the /Default entry of a type-5 halftone dictionary.

void( * DoneHalftone)(const struct sw_htm_dohalftone_request *request, sw_htm_result result)

The screening module calls this to signal to the RIP that the module has finished processing the request.

The result argument indicates the success or otherwise of the halftone processing.

Note:
After calling this function, none of the buffers or other structures referred to by this request should be touched by the module - they all revert to being the exclusive property of the RIP.
Parameters:
[in] request Pointer to the structure detailing the request.
[in] result The value SW_HTM_SUCCESS if halftoning was successful, otherwise one of the sw_htm_result error constants.
sw_htm_raster_ptr* dst_channels

The array of raster channel pointers to the destination channel(s).

sw_htm_coord first_line_y

The position of the band in the overall raster, expressed as the line-index (or y coordinate) of the first line in the band.

sw_htm_raster_ptr msk_bitmap

The pointer to the mask bitmap.

sw_htm_mask_hint msk_hint

This hint is one of the SW_HTM_MASKHINT_* constants. It is intended to allow some optimisation when the mask has all of its bits set, or all of its bits clear.

unsigned int num_dst_channels

The number of raster channel pointers in the dst_channels array.

unsigned int num_lines

The number of raster lines to be halftoned (for each channel). This may be less than the overall band height, such as in the last band of the page or frame.

unsigned int num_src_channels

The number of raster channel pointers in the src_channels array.

sw_htm_raster_ptr object_props_map

The pointer to the object properties map. This will be NULL unless the Boolean want_object_map was set to TRUE in the sw_htm_halftone_details for the halftone instance.

const sw_htm_render_info* render_info

Overall information about the render.

unsigned int separation_number

The ordinal number of the separation being rendered. This corresponds to the separation field of sw_htm_colorant_info. For composite pages, this will always be 1.

sw_htm_raster_ptr* src_channels

The array of raster channel pointers to the source channel(s).

unsigned int thread_index

Index of the current thread, an int in 0...max_render_threads-1.
