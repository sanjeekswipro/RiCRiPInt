Detailed Description

Wrapper for windows.h.

HQNc-standard!pc:export:hqwindows.h(EBDSDK_P.1)

Copyright (C) 2007-2012 Global Graphics Software Ltd. All rights reserved. Global Graphics Software Ltd. Confidential Information.

Its use is to resolve clashes between #defines common to the WDK compiler's headers and our headers, and to turn off warnings which occur in MS Windows include files (winnt.h and winbase.h, for example).

The warnings are:

warning C4201: nonstandard extension used : nameless struct/union warning C4214: nonstandard extension used : bit field types other than int warning C4115: named type definition in parentheses warning C4514: unreferenced inline function has been removed
