Detailed Description

An External Retained Raster interface Page Define event. Note that the memory referred to in this structure should be treated as transient: if the code associated with a handler wishes to use any data contained in this event after the handler has returned, it should make a copy.

Field Documentation

void* connection

Opaque handle to the connection to which this event is relevant. All handlers whose connection handle doesn't match should immediately return SW_EVENT_CONTINUE.

uint32 page_count

The number of pages being defined.

RR_PAGE_DEFINE_PAGE* pages

Array of pages.

sw_tl_ref timeline

Identifies the job containing the pages.
