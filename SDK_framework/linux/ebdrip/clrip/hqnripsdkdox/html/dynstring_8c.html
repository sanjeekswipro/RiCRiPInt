Detailed Description

Dynamic string implementation based on MFS code.

This code uses MFS files to represent strings of arbitrary length. When a new string is added a file will be created in the root of the virtual file system, identified with a unique name (See PSSetStringName()). Any characters added to the string will be stored as bytes inside that file. The file will exist for the lifetime of its corresponding PSString object, being deleted when PSStringClose() is called.
