Detailed Description

Describes a single document within an XPS package.

Field Documentation

uint32 fHasPrintTicket

Flag indicating whether this document has its own print ticket.

Set this to TRUE if this document has its own associated print ticket stream, otherwise set it to FALSE. The RIP uses this flag to determine whether it should call out to the XPSPackageStreamManager with a resource type of ResourceType_Document_PT for this document. If the flag is set to FALSE, the RIP will not make such a call for this document. This can improve efficiency, and avoid the need to implement empty streams where print tickets are not present.

uint8* pszFixedDocumentUri

The absolute URI of the FixedDocument part itself.

If this is NUL, the URI of the document is assumed to follow the conventional XPS package structure where, for instance, the URI for document 1 would be "/Documents/1/FixedDocument.fdoc". If the managed package is using a different structure, the caller should always use this field to override the default. Failure to do this can cause problems when there are relative resource references within the document markup.

uint8* pszPrintTicketUri

The absolute URI of the Print Ticket part associated with this document, if any.

If fHasPrintTicket is FALSE, then this field will be ignored, and would conventionally be set to NUL by the caller. Otherwise, if this field is NUL, the RIP will assume a default URI for the Print Ticket, appropriate to the scope of the document. If a specific Print Ticket URI is known to the caller, it should always be supplied in this field.
