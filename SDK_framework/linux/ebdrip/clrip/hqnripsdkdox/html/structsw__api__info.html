Detailed Description

Required first member for all APIs.

Field Documentation

const uint8* display_name

NUL-terminated UTF-8 string, possibly localised for GUI display. The core RIP does not use this value.

size_t instance_size

The size that the RIP will allocate to construct an instance of the API implementation in which the sw_api_info structure is embedded.

const uint8* name

NUL-terminated internal name used for configuration. Internal names of registered modules must be unique within each module type.

sw_api_version version

The version number of the API.
