SWMSG_RR_ELEMENT_QUERY Struct Reference

#include <swevents.h>

Data Fields

sw_tl_ref timeline

Detailed Description

Looks for a raster element with the given id. Returns SW_EVENT_CONTINUE if nothing found, in which case handle is undefined. Note that NULL is a valid value for handle which means that the element is defined, but is degenerate.

Field Documentation

void* connection

Opaque handle to the connection to which this event is relevant. All handlers whose connection handle doesn't match should immediately return SW_EVENT_CONTINUE.

uintptr_t handle

Element handle - an opaque reference to the cache instance, if any. This will not be used by the RIP directly if external retained raster is on.

uint8* id

Unique ID, length RR_ELEMENT_ID_LENGTH.

sw_tl_ref timeline

Identifies the job containing the pages.
