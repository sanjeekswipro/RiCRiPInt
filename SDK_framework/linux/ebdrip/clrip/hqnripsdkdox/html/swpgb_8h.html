Typedef Documentation

typedef int32 sw_pgb_special_handling

Special handling options for colorants.

Note that any consumer which doesn't understand these will just not apply any special handling, which is correct. Furthermore any consumer which _is_ aware of the specialHandling field but sees a value it doesn't recognise, should just not treat the colorant specially.
